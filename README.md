# ✊ Rock Paper Scissors AI

> A real-time hand gesture Rock Paper Scissors game powered by Computer Vision.
<img width="1536" height="326" alt="banner" src="https://github.com/user-attachments/assets/30739c2d-de28-4b75-943b-d9325fb0391d" />




## Features

- Real-time hand gesture recognition
- AI opponent
- MediaPipe hand tracking
- OpenCV image processing
- Five-round game mode
- Scoreboard
- Winner announcement

---
## Built With

- Python
- OpenCV
- CVZone
- MediaPipe
- NumPy

---

## Installation

```bash
git clone https://github.com/forger234/rock-paper-scissors-ai.git

cd rock-paper-scissors-ai

pip install -r requirements.txt

python main.py
```
